HTML and CSS

HTML

HyperText Markup Language

  • Is used to provide the main structure and content of a website.
  • Uses tags.
  • The lastest version is HTML 5.

CSS

Cascading Styles Sheets

  1. Decorates the page.
  2. Tags can be targetted and then styled. Classes and IDs can also be created for targetting.
  3. 3 different ways to implement - inline, internal, and external.
